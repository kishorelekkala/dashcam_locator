# Dashcam Locator (QGIS Tool)

## Features

- Open dashcam footage directly from the QGIS map  
- Automatically snaps to the nearest recorded route  
- Smooth frame navigation using slider and play/pause  
- Saves selected paths so setup is needed only once  
- Easily switch datasets using the toolbar option  

## Usage

1. Run the script in the **QGIS Python Console**  
2. Click the **camera icon** in the toolbar  
3. On the first run, select:
   - the `geolocated_videos.html` file  
   - the frames root folder  
4. Click near any recorded route on the map to open the dashcam frames
